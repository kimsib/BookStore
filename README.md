# portfolio_bookstoreWeb
미니 쇼핑몰 개인 프로젝트입니다.

# 사용 기술스택
- java 11
- Spring Boot
- Spring Security
- Spring Data Jpa
- Querydsl
- Thymeleaf
- Mysql

# 구현 기능
+ 회원(Member)
  + 회원가입/ 로그인 및 로그아웃

+ 상품(Item)
  + 상품 등록 / 상품 관리 / 상품 수정 / 상품조회(메인화면) / 상품 상세 페이지

+ 주문(Order)
  + 상품 주문 / 구매 이력 조회 / 주문 취소

+ 장바구니(Cart)
  + 장바구니 담기/ 장바구니 조회 / 장바구니 삭제 / 장바구니 상품 주문
