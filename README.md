# 프로젝트 개발 환경
1. 운영체제 : Window
2. 통합개발환경(IDE) : IntelliJ
3. JDK 버전 : JDK 11
4. 스프링 부트 버전 : 2.5.5
5. 데이터 베이스 : MySQL
6. 빌드 툴 : Maven
7. 관리 툴 : Git, GitHub

# 사용 기술스택
- 프론트엔드</br>
HTML, CSS, JS, BootStrap, Thymeleaf</br></br>
- 백엔드</br>
Spring Boot, Spring Security, Spring Data JPA</br></br>
- 데이터베이스</br>
Hibernate, MySQL

# 구현 기능
+ 회원(Member)
  + 회원가입/ 로그인 및 로그아웃

+ 상품(Item)
  + 상품 등록 / 상품 관리 / 상품 수정 / 상품조회(메인화면) / 상품 상세 페이지

+ 주문(Order)
  + 상품 주문 / 구매 이력 조회 / 주문 취소

+ 장바구니(Cart)
  + 장바구니 담기/ 장바구니 조회 / 장바구니 삭제 / 장바구니 상품 주문
